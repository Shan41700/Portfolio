Animations

"Cloud Car"

This animation was inspired by the song 'Cary Me Back' by Mild High Club. I created the illustration in Adobe Illustrator and animated it using Adobe After Effects.

"Sunset Bridge"

This animation is based on my view of the bay at my job. Created in Illustrator, animated with After Effects.

"Disappointment"

The goal of this project was to create an animation that represented an emotion, I chose disappointment. This animation was created using Adobe After Effects.

"An Abduction"

This animation was inspired by the song "The Alien" by Manchester Orchestra. Created in Illustrator, animated with After Effects.

"Duck Amuck"

This piece was inspired by one of my favorite Looney Tunes episodes of all time: Duck Amuck. I created all the graphics using Adobe Illustrator and I used After Effects to animate it.

Graphic Design

"The Heart"

This graphic was created for a social media awareness project that was campaigning for Net Neutrality. This was a piece from a collection of three different pieces all highlighting the importance of having a free and equal internet for all. This graphic was created using Adobe Illustrator.

Travel Posters

For this project I was tasked with creating posters that represented places. I chose my hometown of Rockaway, New York and my college town of Savannah, Georgia. This image shows the two posters placed in a 'real' enviornment.

Designing Chocolate

This is an example of a package design for a chocolate bar. The goal of this project was to create three unique flavors and three unique package designs connected by a central theme. I modeleded my chocolates after the first three The 1975 albums!

"SAM Portrait Postcard"

For this project, I was tasked to create a promotional poster for a fictional art exhibit. This poster is a self-portrait styled in the vaporwave aesthetic and was created using Illustrator. As part of the fictional promotion, the promotional poster had to be placed within context. So I chose to edit my SAM Portrait onto a postcard using Photoshop.

"Horror Flash"

This is a series of graphics I created, inspired by some of my favorite classic horror films, presented in the style of a tattoo artist’s flash tattoo sheet. All graphics were created using Adobe Illustrator.

Paintings

"Still Life #2"

This painting was created as a final project for my honors painting class. This is an acrylic painting on a 16x20 inch canvas. This painting was featured at the Spring Student Art Show at Borough of Manhattan Community College.

"White Flowers"

This is an acrylic still life painting on an 8x10 inch canvas.

"Susan"

This piece was my final project for my Painting II class. For this project, we worked with a model, Susan, who posed in her own home. I was tasked with painting the model and her surroundings, all live. This is an acrylic painting on a 24 x 30 canvas.

"Puppet Zelda"

This is a portrait of Puppet Zelda, a character featured in the video game The Legend of Zelda: Twilight Princess. This is an acrylic painting on a 9x11 canvas.

"Shannon Nouveau"

This 8x10 inch acrylic painting was created as a project for an art history class. It features a self portrait in a classic art nouveau style, defined by it's vintage feel, naturalistic forms, and asymmetry.

Web Design

"The 1975"

This website was created as a midterm project for my Introduction to Web Design class. For this project, I designed a possible website for the band “The 1975”. I then built this website using HTML 5 and CSS. This website is also mobile responsive. The website is deployed on an open-source site hosting service called Github.
